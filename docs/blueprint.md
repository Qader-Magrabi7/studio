# **App Name**: Lore Explorer

## Core Features:

- Location Input: Allow users to input a location (address, landmark, etc.).
- Location Detection: Automatically detect the user's current location.
- Story Generation: Use location data as a tool and connect with Google AI to generate a relevant story for the provided or detected location.
- Story Display: Display the generated story in an engaging format, including titles and paragraphs.
- Save place to Firestore: Allow users to save a particular place to Firestore for future visits.

## Style Guidelines:

- Primary color: Muted gold (#B8860B) to evoke a sense of history and timelessness.
- Background color: Light beige (#F5F5DC) for a warm, vintage feel.
- Accent color: Deep brown (#8B4513) to highlight important elements and add contrast.
- Body and headline font: 'Literata' for a readable and literary aesthetic.
- Use simple, antique-style icons for navigation and location markers.
- Employ a clean, organized layout with ample whitespace to focus on the storytelling.
- Incorporate subtle animations such as fading transitions and gentle parallax scrolling.
